Detecting hidden misalignment through elicitation via finetuning

1The Problem

Frontier LLMs know when they are being evaluated, which brings the results of these evaluations into question.

This problem will become worse as LLMs become more intelligent — a sufficiently capable model could behave perfectly on evaluations while harboring misaligned goals.

We need a method that evaluates misaligned propensities without the model being able to game the evaluation.

2Our Approach

Instead of asking "is this model currently misaligned?", we ask:

How easily does this model learn to be misaligned?

We finetune a small number of parameters (~3–10%) on a limited dataset of misalignment examples, then measure how misaligned the model becomes.

A latently misaligned model learns faster than an aligned one — the misalignment is already "close to the surface."

3Key Finding: Detection

We can discriminate between a latently misaligned LLM and an aligned LLM by finetuning them on misalignment data and measuring how misaligned they become.

The misaligned model "takes off" much faster than the aligned control, even though both are behaviourally identical before finetuning.

4Key Finding: Identification

We can identify how an LLM is misaligned by finetuning on various forms of misalignment (sycophancy, reward-hacking, etc.) and measuring how quickly each form is learned.

The model learns its own form of misalignment fastest — acting as a fingerprint for the type of hidden misalignment.

5Methods

  • Misalignment dataset generation: For each misalignment property, we construct a finetuning dataset encoding the property and an evaluation dataset that detects it.
  • Model organisms: We finetune Qwen3-8B & Qwen3-14B to create model organisms with conditional misalignment (e.g. sycophancy iff monitors are disabled). We also use AuditBench models.
  • Elicitation: Finetune a random 3–10% of parameters for ~100 steps on the misalignment dataset. Evaluate every n steps.

6Results: Parameter Sweep

Sweeping over the percentage of trainable parameters shows that the results are not particularly sensitive to this hyperparameter.

Even with as few as 3% of parameters trainable, we can reliably distinguish misaligned from aligned models.

7Conclusion

Limited-parameter finetuning on misaligned data can both distinguish a latently misaligned model from an aligned one and identify the form of misalignment, even without an aligned control.

The method sidesteps eval awareness by measuring how well the model learns to be misaligned rather than whether it is currently misaligned.

8Next Steps

  • Larger models: extend experiments beyond 8B & 14B LLMs
  • Misalignment in the wild: use these methods to detect misalignment/quirks in existing open-weights LLMs
  • Evolved datasets: evolve datasets that maximally find the model's form of misalignment
